Introduction

Information flow. A system is noninterfering if the low security outputs of the system are independent of the high security inputs, and thus, an observer with low security clearance cannot deduce any information about high security data. When information originating from different principals is combined, the information flow restrictions of all principals must be enforced.

Access control. the owner of a file can specify an access control list--a list of principals who may read the resource.

Software licensing. Software producers distribute software components under licenses that include restrictions on when the software may be composed with other components to form new software.
